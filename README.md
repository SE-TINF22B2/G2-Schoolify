# Getting Started

## Frontend
To start the **frontend**: 
- navigate to folder clientside
- run `npm run dev`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Backend
To start the **backend**: 
- navigate to folder serverside
- run `npm run dev`
